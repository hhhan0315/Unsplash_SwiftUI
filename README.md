# Unsplash_SwiftUI
Unsplash Image API를 활용한 사진 앱
Unsplash unofficial app for iOS

> Network - Secrets - clientID에 Unsplash API Key 입력 후 실행

> iOS 15.0

## 기술 정보
- 화면 구현 : SwiftUI
- 네트워킹 : URLSession
